Learn HTML

Welcome to your journey of learning HTML! In this guide, you’ll learn everything you need to start building websites using HTML — the language of the web.

What is HTML?

HTML stands for HyperText Markup Language. It’s not a programming language — it’s a markup language used to create the structure of web pages.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

HTML Structure

Every HTML page has this basic structure:

  • <!DOCTYPE html>: Tells the browser you’re using HTML5.
  • <html>: The root of your page.
  • <head>: Contains the page title, meta info, links to CSS.
  • <body>: This is where all your visible content goes.

Headings

HTML has 6 heading tags:

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Smaller Heading</h3>
... up to <h6>

Paragraphs and Text

<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>
<br> (line break)

Links

<a 
href="https://example.com">Visit
 
Example</a>

Images

<img src="image.jpg" alt="Description">

Lists

Unordered List:


<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Ordered List:


<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

Tables


<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Israel</td>
    <td>22</td>
  </tr>
</table>

Forms


<form>
  <label>Name:</label>
  <input type="text" name="name">
  <input type="submit" value="Send">
</form>

HTML Comments


<!-- This is a comment -->

HTML Entities

Use these to show special characters:

  • &lt; = <
  • &gt; = >
  • &amp; = &

Semantic Tags

These tags give meaning to your layout:

  • <header> – for page header
  • <nav> – navigation links
  • <main> – main content
  • <article> – blog post or article
  • <footer> – page footer

Putting It All Together

Here’s a full mini page you can copy:


<!DOCTYPE html>
<html>
  <head>
    <title>Tech Life HTML Page</title>
  </head>
  <body>
    <header>
      <h1>Welcome to Tech Life</h1>
    </header>
    <main>
      <p>We teach you how to code!</p>
    </main>
    <footer>
      <p>© 2025 Tech Life</p>
    </footer>
  </body>
</html>
