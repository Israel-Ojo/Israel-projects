learn CSS

CSS (Cascading Style Sheets) is used to control the layout and appearance of web pages. Here's a comprehensive guide to get you started:
CSS is used to separate presentation from structure, making it easier to maintain and update websites. You can use CSS to control the layout, colors, fonts, and more.
*CSS Basics* - Selectors: Used to target specific elements on a web page. - Properties: Used to define the styles applied to selected elements. - Values: Used to specify the value of a property.
*CSS Selectors* - Element selectors (e.g., `h1`, `p`, `img`) - Class selectors (e.g., `.header`, `.footer`) - ID selectors (e.g., `#logo`) - Attribute selectors (e.g., `[hreflang]`)
*CSS Properties* - `color`: Specifies the color of text. - `background-color`: Specifies the background color of an element. - `font-size`: Specifies the size of text. - `font-family`: Specifies the font family of text. - `margin` and `padding`: Specify the spacing between elements.
*CSS Units* - `px`: Pixels - `em`: Relative to the font size of the element - `rem`: Relative to the root element's font size - `%`: Percentage
*CSS Box Model* - `width` and `height`: Specify the size of an element. - `padding`: Specifies the space between the content and the border. - `border`: Specifies the border of an element. - `margin`: Specifies the space between elements.
